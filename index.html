ro-repo Fedora Paket Deposu

Açık Kaynak Geliştirme Topluluğu tarafından sunulan resmi Fedora RPM paket deposuna hoş geldiniz. Bu rehber, repoyu sisteminize nasıl ekleyeceğinizi ve paketleri nasıl kuracağınızı açıklamaktadır.

1. Depoyu Sisteme Eklemek

Aşağıdaki komutu kopyalayarak terminalinize yapıştırın. Bu komut, hem 64-bit (x86_64) hem de mimariden bağımsız (noarch) paketler için gerekli depo yapılandırmasını sisteminize ekleyecektir:

sudo tee /etc/yum.repos.d/ro-repo.repo <<'EOF'
[ro-repo]
name=Acik Kaynak Gelistirme Toplulugu Repo
baseurl=https://project-ro-asd.github.io/Ro-Repo/$basearch/
enabled=1
gpgcheck=0

[ro-repo-noarch]
name=Acik Kaynak Gelistirme Toplulugu Repo - Noarch
baseurl=https://project-ro-asd.github.io/Ro-Repo/noarch/
enabled=1
gpgcheck=0
EOF

2. Önbelleği Güncellemek

Depoyu ekledikten sonra, sisteminizin yeni paketleri algılayabilmesi için DNF önbelleğini temizleyin ve güncelleyin:

sudo dnf clean all
sudo dnf makecache

3. Paket Kurulumu

Artık depomuzdaki paketleri kurmaya hazırsınız! Repodaki mevcut paketleri listelemek için aşağıdaki komutu kullanabilirsiniz:

dnf repo-pkgs ro-repo list

Bir paketi kurmak için standart DNF komutunu kullanın (örneğin ro-control paketi için):

sudo dnf install ro-control
